# Read Me
* news.csvをsummarize0.jsがゴミ除去整理してnews1.csvとnews2.csvに分割
* summarize1.jsがnews1.csvを読んでsummary1.txtに出力
* summarize1.5.jsはnews2.csvを読んでsummary1.txtに出力を追記
* summarize2.jsがsummary1.txtを並べ替えてsummary2.txtを出力
* summarize3.jsはsummary1.txtからワードクラウド用の重要度ランキングをsummary3.txtに出力する。
* summarize4.jsはsummary2.txtを英文翻訳記事補充など最終確認してsummary2.txt に出力。warehouseフォルダにyyyyMMddHHmm.textを保存。
* index.htmlのニュースはsummary2.txtのものを表示する。
